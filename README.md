#  Интернет-магазин книг
Простой код сайта для книжного магазина.

## Установка
1. Сохраните все папки и файлы проекта в главную папку
2. Создайте в главной папке файл .env и укажите в нем желаемые значения параметров настроек для базы данных из файла config.py
   Пример файла .env:
   ```python
SECRET_KEY=1234
DATABASE_URL='sqlite:///database.db'
```
3. Установите в окружение необходимые библиотеки из файла requirements.txt
4. Запустите файл app.py

## Использование
- Чтобы добавить книгу в базу данных, используйте функцию add_book() в файле routes.py
- Чтобы добавить в базу данных несколько книг из файла json, используйте функцию load_from_file() в файле routes.py
